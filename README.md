## 🌟 Overview

AI Recipe Generator is a **Full Stack MERN Application** that transforms food images into complete recipes using AI.

Users can upload a food image, generate detailed recipes with cooking instructions, nutrition information, save their own recipes, and securely manage their accounts through JWT authentication.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/anjalipatel-tech/AI-Recipe-Generator.git
```

Move into project

```bash
cd AI-Recipe-Generator
```

---

## Install Frontend

```bash
cd client
npm install
npm run dev
```

---

## Install Backend

```bash
cd server
npm install
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGO_URI=Your_MongoDB_URI

JWT_SECRET=Your_JWT_SECRET

OPENROUTER_API_KEY=Your_API_Key
```
